# ⛅ Weather Telegram Bot (with Ansible Deployment)
Бот для получения погоды, упакованный в Docker. В репозитории также находится Ansible-плейбук для автоматического развертывания на любом облачном сервере Ubuntu за 2 минуты.
## 🚀 Быстрый старт (Локально через Docker)
1. Склонируй репозиторий.
2. Создай файл `.env` и добавь туда переменные `TG_TOKEN` и `WEATHER_API_KEY`.
3. Запусти: `docker-compose up -d`.
